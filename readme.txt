## Prérequis :

    - Un éditeur de code (VS Code /CLion /Sublime Texte...)

    - Installation de MINGW

    - si vous utilisez un editeur de texte autre que clion rendez vous dans la fonction extractWordByTypeInDictionnary ( dans file.c)
    - et supprimez les "../" qui se trouve sur la meme ligne que Dico

## Comment exécuter le projet ?

1 - lancer la commander :  """ git clone https://github.com/lionelmarcus10/SentenceGen """ sur votre terminal

- ou copier le projet


2 - Ouvrir le projet dans un éditeur de code , selectionner l'interpreteur C


3 - Executer le fichier "main.c"
